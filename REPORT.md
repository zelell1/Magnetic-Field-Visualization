# Отчет по вычислительному практикуму

**Тема:** расчет и визуализация магнитного поля цилиндрического соленоида методом суперпозиции полей отдельных витков.

## 1. Цель и постановка задачи

Требуется вычислить и визуализировать магнитное поле цилиндрического соленоида диаметра `D`, длины `L` и числа витков `N`. Поле строится в плоскости `(x, y)`, содержащей ось соленоида. Ось соленоида направлена вдоль `x`, поперечная координата в выбранной плоскости обозначена `y`.

Параметры должны задаваться пользователем. В работе реализовано интерактивное демо, где можно изменять:

- диаметр `D`;
- длину `L`;
- число витков `N`;
- ток `I`;
- размер расчетной области;
- разрешение сетки;
- число расчетных витков, используемых для численной суперпозиции;
- отображаемую величину: `|B|`, `Bx` или `By`.

## 2. Физическая модель

Соленоид моделируется как система круговых токовых витков радиуса

```text
R = D / 2.
```

Центры витков лежат на оси `x` внутри отрезка `[-L/2, L/2]`. Магнитное поле находится по принципу суперпозиции:

```text
B(x, y) = sum_i B_i(x, y),
```

где `B_i` -- поле `i`-го кругового витка. Такой подход физически корректен для стационарных токов, поскольку уравнения магнитостатики линейны по току.

Для большого `N` соседние витки можно группировать в расчетные элементы с весом `w_i`. При этом сохраняется суммарное число ампер-витков:

```text
sum_i w_i I = N I.
```

Это ускоряет интерактивную визуализацию и не меняет интегральную физическую модель.

## 3. Поле одного кругового витка

Для одного витка используется аналитическое выражение через полные эллиптические интегралы. Пусть

```text
xi  = x - x_i,
rho = |y|,
alpha^2 = (R + rho)^2 + xi^2,
beta^2  = (R - rho)^2 + xi^2,
m = 4 R rho / alpha^2.
```

Тогда компоненты поля в цилиндрических координатах:

```text
Bx = mu0 I / (2 pi alpha) *
     [ K(m) + (R^2 - rho^2 - xi^2) / beta^2 * E(m) ],

Br = mu0 I xi / (2 pi rho alpha) *
     [ -K(m) + (R^2 + rho^2 + xi^2) / beta^2 * E(m) ].
```

В выбранной плоскости `By = sign(y) Br`. На оси `rho = 0` используется предельная формула:

```text
Bx = mu0 I R^2 / (2 (R^2 + xi^2)^(3/2)),
By = 0.
```

Полные эллиптические интегралы вычисляются численно:

```text
K(m) = integral from 0 to pi/2 of dtheta / sqrt(1 - m sin^2(theta)),
E(m) = integral from 0 to pi/2 of sqrt(1 - m sin^2(theta)) dtheta.
```

В программе эти интегралы считаются квадратурой Гаусса-Лежандра.

## 4. Численный алгоритм

Алгоритм расчета:

1. Пользователь задает `D`, `L`, `N`, `I` и параметры сетки.
2. Вычисляется радиус `R = D / 2`.
3. Формируется набор расчетных витков с координатами `x_i` и весами `w_i`.
4. Для каждой точки сетки `(x, y)` считается поле каждого расчетного витка.
5. Вклады суммируются:

```text
Bx(x, y) = sum_i Bx_i(x, y),
By(x, y) = sum_i By_i(x, y).
```

6. На Canvas строится цветовая карта `|B|`, `Bx` или `By`, поверх нее рисуются стрелки направления поля и геометрия соленоида.
7. Дополнительно строится график `Bx(x, 0)` на оси соленоида и сравнивается с аналитической формулой конечного соленоида.

## 5. Проверочные аналитические выражения

Для непрерывного конечного соленоида с плотностью витков

```text
n = N / L
```

поле на оси равно:

```text
Bx_axis(x) = mu0 n I / 2 *
             [ (x + L/2) / sqrt(R^2 + (x + L/2)^2)
              -(x - L/2) / sqrt(R^2 + (x - L/2)^2) ].
```

В центре длинного соленоида:

```text
B0 ~= mu0 N I / L.
```

В дальней зоне поле на оси стремится к дипольной асимптотике:

```text
Bx(x) ~= mu0 N I R^2 / (2 |x|^3).
```

Эти формулы используются для контроля корректности физики и для автоматических тестов.

Проект написан на HTML, CSS и JavaScript без внешних npm-зависимостей.

- `index.html` -- структура интерфейса;
- `styles.css` -- визуальное оформление;

## 7. Проверка корректности

Автоматические тесты проверяют:

- предел эллиптических интегралов при `m = 0`: `K(0) = E(0) = pi / 2`;
- совпадение поля одного кругового витка на оси с аналитическим выражением;
- симметрию поля относительно оси: `Bx(x, y) = Bx(x, -y)`, `By(x, y) = -By(x, -y)`;
- приближение поля длинного соленоида к `mu0 N I / L`;
- совпадение дискретной суперпозиции с формулой конечного соленоида на оси;
- дипольную асимптотику в дальней зоне;
- изменение направления поля при смене знака тока.

Результат запуска:

```text
tests 7
pass 7
fail 0
```

## 8. Пример результата

Для базовых параметров демо

```text
D = 0.20 m,
L = 0.80 m,
N = 120,
I = 2.0 A
```

получается:

```text
Bx(0, 0) по суперпозиции = 365.74 microT,
Bx(0, 0) по формуле конечного соленоида = 365.74 microT,
B длинного соленоида = 376.99 microT.
```

Отличие от формулы длинного соленоида связано с конечной длиной: у реального конечного соленоида поле у торцов рассеивается, поэтому центральное поле немного меньше предельной оценки `mu0 N I / L`.

## 9. Обсуждение результатов

Внутри соленоида поле направлено преимущественно вдоль оси `x` и при достаточно большом отношении `L/D` становится почти однородным. Вблизи торцов появляются заметные поперечные компоненты поля, линии поля выходят наружу и замыкаются вне соленоида. При удалении от соленоида поле быстро уменьшается и соответствует дипольной асимптотике.

Интерактивное изменение параметров показывает ожидаемые зависимости:

- увеличение тока `I` линейно увеличивает поле;
- увеличение числа витков `N` также линейно увеличивает поле;
- при увеличении длины `L` при фиксированном `N` уменьшается плотность витков `n = N / L`, поэтому центральное поле становится меньше;
- при увеличении `L/D` область почти однородного поля внутри соленоида расширяется.

## 10. Выводы

В работе реализована физически корректная модель магнитного поля цилиндрического соленоида на основе принципа суперпозиции. Поле одного витка вычисляется по формулам Биота-Савара в форме с эллиптическими интегралами, а поле соленоида получается суммированием вкладов витков.

Программа содержит интерактивную визуализацию, управляемые параметры, сравнение с аналитическим осевым решением и минимальный набор автоматических тестов. Полученные результаты согласуются с известными предельными случаями: формулой длинного соленоида в центре и дипольной асимптотикой в дальней зоне.
